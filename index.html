ElCoin uses SHA-256 Proof of Work and ECDSA (P-256) digital signatures — every transfer is cryptographically signed and verified.
Each block references the previous block's hash, forming an immutable chain. Tampering invalidates the whole chain.
This runs on a single local node in your browser. A real cryptocurrency needs a peer-to-peer network of many independent nodes — see the README for the real on-chain path.
